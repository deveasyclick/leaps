# Tilvio

Admin dashboard for uploading study resources for leaps mobile app

🔗 **Live Demo:** [https://leaps-uk.web.app](https://leaps-uk.web.app)

## 🚀 Features

- Upload and manage resources
- Admin dashboard for uploading resources
- Responsive UI for desktop and mobile

## 🛠️ Tech Stack

**Frontend:** React, Redux, 
**Backend:** Nodejs, Typescript, Express, Firebase functions 
**Database:** Firebase firestore
**Deployment:** firebase
**Others:** Material UI

## 🧑‍💻 Getting Started

### Prerequisites

- React 16+
- Node.js 10+
- Firebase

### Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/tilvio.git
   cd tilvio
   ```
2. Copy .env.example to .env and set values:
   ```
   cp .env.example .env
   ```
3. Run the development servers:
   ```bash
   npm run dev
   ```
4. Project Structure
   ```bash
   leaps/
   ├── public/
   ├── src/     
   │   ├── assets/
   │   └── components/
   │   └── config/
   │   └── helpers/
   │   └── layouts/
   │   └── pages/
   ├── .eslintrc.json
   └── .gitignore
   └── .firebase.json
   └── .package.json
   └── .package-lock.json
   ```
